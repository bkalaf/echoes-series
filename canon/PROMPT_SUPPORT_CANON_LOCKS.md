# Prompt Support Canon Locks

Centralized prompt-generation locks for spelling, naming, character usage, location usage, and recurring “use X, not Y” corrections.

This file exists so chapter prompts do not have to carry correction clutter. Chapter and vision files should state the correct form directly and link to this file as required canon when needed.

---

# How To Use This File

Use this file as a required attachment for prompt-ready chapter, vision, and interlude generation packets.

A chapter prompt may include chapter-specific constraints, but it should not repeat old correction logs unless the correction is still unresolved and directly relevant to that chapter.
