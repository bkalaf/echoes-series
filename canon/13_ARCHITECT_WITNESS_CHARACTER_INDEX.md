## Use Rules

- The First Mirror council members should be called **council members** in Addie-facing Book 01 prose, not Architects.
- “Architect” is an author/canon term and later-series reveal term.
- Accent notes are performance/voice cues, not phonetic-spelling instructions. Do not write heavy dialect unless a later dedicated voice pass locks it.
- Ethnicity/culture notes are provisional Earth-origin coding unless explicitly marked locked.
- Visual descriptions are intentionally scaffolded. Replace / refine them after image upload and image-parsing pass.
- Witness-form descriptions should support iconography, scene blocking, and image generation, but the human council form remains primary for First Mirror prose.

## Visual Pass Pending

When reference images are uploaded, parse each image into:

- face shape / age read;
- hair color, hair texture, style;
- skin tone and visible ethnic coding;
- body type / posture;
- clothing silhouette;
- signature object or symbol;
- expression baseline;
- what should never be changed in future prompts;
- what should remain flexible.

Then update the roster above with image-derived visual anchors.

## Accent Pass Pending

For each character, decide whether accent should be:

- neutral / unmarked;
- lightly region-coded in rhythm only;
- formally noted for performance direction;
- never rendered phonetically.

Default rule: use rhythm, word choice, and sentence structure instead of phonetic spelling.

## First Mirror Usage Note

In Book 01 First Mirror prose, the reader should hear one recoverable value-line from every council member, but the full later compromise logic remains hidden / pressure-obscured.

The goal is rereadability:

> Later readers should realize every council member already named the exact value they would betray.
