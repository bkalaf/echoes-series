# Canon Priority

Use this hierarchy whenever sources conflict.

## Source-of-Truth Order

1. Latest explicit user correction.
2. Current canon files in this repository.
3. Current book matrix / book spine.
4. Character registry.
5. Witness / Architect / Reward registry.
6. Chapter outlines.
7. Older drafts.
8. Brainstorming notes.

## Hard Continuity Rule

Do not solve a downstream issue by dropping a foundational requirement. If a new change conflicts with earlier canon, flag the conflict and preserve the newest explicit correction.

## Regression Checklist

Before accepting a new canon update or chapter draft, verify:

- Names match current canon.
- Companion, Witness, Reward, and book number match the book spine.
- Beacon cities and phase colors are preserved.
- Ship status is not overwritten.
- One-nation / city-state structure is preserved.
- Chapter section breaks are preserved when marked.
- Reconstruction-grade outlines preserve beat order and ending state.
