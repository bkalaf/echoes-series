# Image Upload Register

This file is the comprehensive register for user-uploaded image references.

It does **not** store image binaries. It records the visual canon derived from the uploaded images so the images can be found conceptually and linked to the correct canon area: characters, Witnesses, settings, chapter scenes, constellations, and reward symbolism.

## Usage Rule

Use this file as the **upload coverage ledger**. Use `14_VISUAL_BIBLE_IMAGE_PASS.md` for synthesized visual rules and prompt direction.

If an image contains readable text, badges, wrong names, wrong ranks, wrong faction banners, or modern-looking artifacts, those details are **not canon** unless confirmed elsewhere. The canon record should preserve the visual intention, not accidental text.
