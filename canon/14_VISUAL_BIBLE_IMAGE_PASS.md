# Usage Rules

- Treat this file as visual guidance, not prose canon.
- Do not override locked names, ranks, ages, species, or continuity because an image contains incorrect text.
- In Book 01 First Mirror prose, call the historical figures **council members**, not Architects.
- Accent notes are performance direction. Use rhythm and diction, not heavy phonetic dialect unless a character has an explicit exception.
- Image text, banners, badges, and labels are noncanonical unless separately confirmed.
- Architect = human/civic/before layer.
- Witness = mythic/symbolic/wounded after layer.
- Constellation = sky-memory layer.
- Story reference = scene mood, staging, and symbolic palette.

# Global Visual Read

## Architect Layer

Architect images should feel rational, civic, human, procedural, hopeful, and coordinated:

- luminous blue-gold civic architecture;
- glass map tables;
- drafting instruments;
- city models;
- cathedral-scale planning halls;
- white, blue, and gold formal robes;
- one unmistakable profession/value object per council member.

Risk: visual sameness. Every council member needs a clear object language.

## Witness Layer

Witness images should feel mythologized, wounded, symbolic, spectral, and domain-corrupted:

- animal or mythic form;
- color-coded spectral domain;
- symbolic environment;
- corrupted value made visible;
- readable silhouette at small scale.

## Story Iconography Layer

Book 01 story images define public-history and civic-myth language:

- Dovren imperial decadence;
- Addie becoming a public symbol by accident;
- Addie’s private self-fashioning;
- zodiac order as calendar / cosmic memory;
- reward constellations rewriting the sky after Witness defeat;
- Royal Observatory and summons scrolls as procedural plot machinery.
