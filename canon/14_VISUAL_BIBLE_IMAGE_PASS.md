# Visual Bible — Image Pass Notes

This file records the image-analysis notes from the first major visual-reference pass.

It does **not** store the images themselves. It stores the canonical visual read derived from the user-provided image batches.

## Source Batches

The notes below come from three image batches supplied in conversation:

1. **Initial character / creature sample set** — companion/support/creature/institutional imagery: Ashwing, Brickett, Brimscale, Brynn, Carrio, Choleth, Tallandra, Geoffrik, Gregory, Hans, Mae, Mattieu, Max.
2. **Architect set** — images named by the eventual Witness identity, but representing the **human / pre-Witness Architect forms**.
3. **Actual Witness set** — mythic/spectral Witness forms.
4. **Additional character images** — expanded Architect/Witness/support visuals supplied afterward, including blue-gold civic planners and several mythic Witness refinements.

## Usage Rules

- Treat these notes as a **visual bible layer**, not as final prose canon.
- Do not overwrite locked names, ranks, species, or roles because an image contains incorrect text.
- For Book 01 First Mirror prose, call the historical figures **council members**, not Architects, unless later-series reveal context allows the author-facing term.
- Ethnicity / cultural-coding notes describe visual inspiration and casting direction, not real-person identity.
- Accent notes are performance direction only. Avoid heavy phonetic dialect.
- If an image contains visible text, treat it as noncanonical unless explicitly confirmed.
- The strongest pattern is: **Architect = rational/civic/human before myth; Witness = symbolic wound after compromise; Constellation = what Eidolon remembers after myth becomes story.**

# Global Visual Read

## Architect Layer

The Architect images work best as the **before** layer:

- luminous blue-gold civic architecture;
- glass map tables;
- drafting instruments;
- holographic city models;
- cathedral-scale planning halls;
- white, blue, and gold formal robes;
- solar/celestial civic motifs;
- “we are building the future” energy.

This is excellent for the complete Book 14 council and the Book 1 First Mirror adaptation because it makes the council feel like brilliant, moral, practical people trying to save civilization.

The risk is visual sameness: many read as “beautiful person in a blue-gold planning room.” Each Architect needs one unmistakable occupational/moral object.

## Witness Layer

The actual Witness images are much more distinct. They should be treated as the primary **mythic iconography** layer:

- animal/mythic form;
- color-coded spectral domain;
- symbolic environment;
- corrupted value made visible;
- boss-encounter / religious-icon / constellation-seed energy.

## Three-Layer Structure

| Stage | Visual Meaning |
|---|---|
| Architect | rational, civic, human, procedural, hopeful, coordinated |
| Witness | mythologized, wounded, symbolic, spectral, domain-corrupted |
| Constellation | what Eidolon remembers after even the Witness has become story |

## Biggest Witness Correction

**Horror / Regret** needs the most emotional repositioning.

Current image is visually powerful but leans too much toward seduction/spider queen. The Book 1 and Book 14 function needs Horror/Regret to feel like:

- emergency evacuation;
- impossible route choices;
- names left behind;
- helper corrupted by what he could not prevent;
- fear turning into regret;
- Ariadne’s Thread as operational salvation and wound.

Better direction:

> black Arachne archivist in a ruined transit hall, holding a glowing thread stretched across evacuation ledgers, empty name tags, abandoned children’s shoes, and broken route maps.

# Prompting / Future Image Generation Rules

## Architect Prompts

Architect prompts should include:

- luminous civic planning hall;
- blue-gold-white palette;
- glass map table or field-specific planning surface;
- formal but human expression;
- pre-collapse council seriousness;
- one unmistakable profession/value object;
- no overt monstrous Witness features unless intentionally foreshadowing.

## Witness Prompts

Witness prompts should include:

- spectral domain color;
- animal/mythic form;
- domain-corrupted environment;
- object tied to original Architect role;
- beauty plus wound, not generic monster design;
- symbolic readability at thumbnail scale.

## Character Prompting Warning

Do not use the actual Witness images as literal First Mirror forms. Book 1 First Mirror sees the human council, with only limited recognition effects such as Solomon/Ash.
