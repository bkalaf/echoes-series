# Maintenance Rule

When a scratchpad planning file introduces or changes a reusable setting or scene:

1. Add/update the setting in [`../canon/15_SETTINGS.md`](../canon/15_SETTINGS.md).
2. Add/update the scene in [`../canon/16_SCENES.md`](../canon/16_SCENES.md).
3. Update this file with the relevant scene IDs and setting IDs.
4. Promote to `drafts/` only when the file becomes a formal chapter outline rather than rough planning.
