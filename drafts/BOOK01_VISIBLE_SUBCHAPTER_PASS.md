# Book 01 Visible Subchapter Consolidation Pass

## Status

Controlling review pass for reader-facing visible subchapter headings across active Book 01 draft outlines.

This file supersedes older over-detailed `Suggested visible subchapters` lists when preparing prompt-ready prose packets. Older beat-by-beat headings inside draft files may remain as **author-facing beat labels** until each packet is rewritten, but they should not be rendered as visible prose subchapter headings unless they appear in the table below.

## Rule

Visible subchapters should mark significant location changes, POV changes, major time jumps, formal inserted structures, or major act-level turns.

Same-room / same-POV tactical beats, jokes, argument turns, or small emotional pivots should use a standard markdown scene break:

```md
---
```

Do not use the Mark of the Orbs on subchapter headings. The Mark appears only in the chapter heading box.
