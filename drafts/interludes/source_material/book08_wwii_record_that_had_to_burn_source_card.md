## Core Function

This interlude teaches that ledgers and records are not passive history. When a state decides a category is disposable, meticulous records become hunting machinery. Forgery can save lives for a while, but if the master registry remains intact, the machine can still reference itself and expose the forgery.

The sabotage therefore becomes both practical and symbolic: destroy the machine so the machine cannot reference the machine.
