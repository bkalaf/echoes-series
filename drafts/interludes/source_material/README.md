## Extraction Rule

Do not treat every upload as final prose. For each book, perform a placement/extraction pass:

1. Confirm target book and theme from `BOOKS.md`, `BOOK_THEMES.md`, and `canon/04_DEAR_READER_INTERLUDES.md`.
2. If the source is near-ready prose, preserve its voice and expand/polish rather than rewriting from scratch.
3. Extract or generate the final interlude draft under `drafts/interludes/`.
4. Add the compile slot or source reference to the relevant master outline / book outline.
5. Mark the source as extracted here.
