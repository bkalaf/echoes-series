## Extraction Rule

Do not treat every upload as final prose. For each book, perform a placement/extraction pass:

1. Confirm target book and theme from `BOOKS.md`, `BOOK_THEMES.md`, and `canon/04_DEAR_READER_INTERLUDES.md`.
2. Extract or generate the final interlude draft under `drafts/interludes/`.
3. Add the compile slot or source reference to the relevant master outline / book outline.
4. Mark the source as extracted here.
