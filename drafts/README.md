# Drafts Index

This directory contains parsed chapter outlines, reconstruction outlines, interlude drafts, and protected draft fragments.

## Crosslink Rule

Every parsed draft should connect to:

- Characters: [`../canon/05_CHARACTERS.md`](../canon/05_CHARACTERS.md)
- Settings: [`../canon/15_SETTINGS.md`](../canon/15_SETTINGS.md)
- Scenes: [`../canon/16_SCENES.md`](../canon/16_SCENES.md)
- Master outline: [`../canon/SERIES_MASTER_OUTLINE.md`](../canon/SERIES_MASTER_OUTLINE.md)

Settings are reusable places: city / section / district / site / chamber.

Scenes are specific narrative units: time/date + setting + cast.

# Maintenance Rule

When a draft adds or changes location, date/time, or cast:

1. Update [`../canon/15_SETTINGS.md`](../canon/15_SETTINGS.md) if a reusable setting changed.
2. Update [`../canon/16_SCENES.md`](../canon/16_SCENES.md) if a scene changed.
3. Update this index if scene IDs or setting IDs changed.
