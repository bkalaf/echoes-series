# Maintenance Rule

When a draft adds or changes location, date/time, or cast:

1. Update [`../canon/15_SETTINGS.md`](../canon/15_SETTINGS.md) if a reusable setting changed.
2. Update [`../canon/16_SCENES.md`](../canon/16_SCENES.md) if a scene changed.
3. Update [`../canon/10_BOOK01_CHAPTER_OUTLINES.md`](../canon/10_BOOK01_CHAPTER_OUTLINES.md) if Book 1 chapter status changes.
4. Update [`../canon/12_BOOK01_CALENDAR_TIMELINE.md`](../canon/12_BOOK01_CALENDAR_TIMELINE.md) if Book 1 date / weekday / subchapter calendar placement changes.
5. Update this index if scene IDs, setting IDs, correction layers, or calendar positions change.
